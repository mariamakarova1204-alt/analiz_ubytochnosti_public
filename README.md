# Анализ убыточности типографии
## Описание проекта
В рамках данного проекта проведён анализ финансово-хозяйственной деятельности типографии за 2024–2025 гг. с целью выявления факторов, влияющих на прибыльность предприятия.
Проект включает этапы подготовки и очистки данных, SQL-обработку, расчёт финансовых показателей, ABC-анализ контрагентов и создание интерактивного дашборда в Power BI.
---
## Цель
Выяыить причины убыточности типографии
---
## Данные
- 1С:Бухгалтерия
---
## Задачи проекта
- выполнить загрузку и предварительную обработку данных;
- провести очистку и подготовку данных для анализа;
- рассчитать показатели выручки, затрат, прибыли и рентабельности;
- выполнить ABC-анализ контрагентов;
- подготовить SQL-запросы для формирования аналитических витрин;
- разработать интерактивный дашборд в Power BI для визуализации результатов анализа.

## Выполненный анализ
### Финансовый анализ
- анализ динамики выручки;
- расчёт выручки без НДС;
- анализ затрат предприятия;
- расчёт прибыли;
- оценка рентабельности деятельности.
### ABC-анализ
Контрагенты были распределены по группам:
- **A** — наиболее значимые контрагенты;
- **B** — контрагенты средней значимости;
- **C** — прочие контрагенты.
---
## Дашборд Power BI
Разработан интерактивный дашборд, включающий:
### Страница 1. Финансовые показатели
- динамика выручки;
- динамика затрат;
- прибыль по месяцам;
- показатели рентабельности;
- сравнение результатов 2024 и 2025 годов.
### Страница 2. Анализ затрат и ABC-анализ
- структура затрат предприятия;
- распределение контрагентов по группам A/B/C;
- ключевые контрагенты по объёму выручки;
- интерактивные фильтры по годам.
